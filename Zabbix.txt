https://www.zabbix.com - на сайте во вкладке download выставляем параметры сервера, существующей БД и существующего веб-сервера.
Если ни БД ни вуб-сервер не установлены, то надо установить.
Выбираем релиз LTS
Далее копируем команды с сайта.

После установки проверим есть ли БД для заббикса:
mysql> show databases;
+--------------------+
| Database |
+--------------------+
| information_schema |
| mysql |
| performance_schema |
| sys |
| zabbix |
+--------------------+

Редактируем конфиг заббикса:
nano /etc/zabbix/zabbix_server.conf
Ищем закомментированную строчку, раскомментируем ее

# DBPassword=задаем*ранее*указанный_пароль

Короче все по шагам на офиц сайте

После всех моментов чтобы зайти надо ввести в браузере адрес_сервера/zabbix
Еще несколько шагов
Юзер: Admin

Меняем пароль Admina во вкладке Administration-Users
